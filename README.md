# ChatGPT-Clone

A conversational AI web application that replicates the core functionality of ChatGPT, allowing users to interact with an AI assistant in real time through a modern chat interface.

## 🚀 Features
- Real-time AI chat interface
- Streaming responses
- Clean and responsive UI
- Built using modern web technologies

## 🛠️ Tech Stack
- Next.js
- React
- Tailwind CSS
- AI SDK (Groq API)

## ▶️ Getting Started

Run the development server:

```bash
npm run dev
